# Kotlin Rust Interop
  - An attempt to implement methods in Kotlin via rust.  
###  RSA algorithm

### Prerequisite
Ensure you have the following installed:
 1. kotlin
 2. Java
 3. Rust

#### Setup
``` 
git clone https://github.com/Wambug/learn_kotlin
```
#### Run
```
make run 
```
